# Overview

The options for the mark down table builder can setup three main areas:

| Behaviour                                               | Notes                                                                               |
| ------------------------------------------------------- | ----------------------------------------------------------------------------------- |
| Value Transformers                                      | These transform an object value into a string                                       |
| [Property metadata provider](./03-default-meta-data.md) | Generates `PropertyMetaData` from a `PropertyInfo` instance                         |
| [Default output](02-output.md)                          | The output that is returned when an empty enumerable is used. It defaults to `null` |
